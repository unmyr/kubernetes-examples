== Install

=== Install KIND

. Set `go` bin path
+
[source,shell]
.~/.bashrc
----
test -d $HOME/go/bin && export PATH="$HOME/go/bin:$PATH"
----

. Apply changes
+
[source,shell]
----
exec $SHELL -l
----

. Install KIND
+
[source,shell]
----
go install sigs.k8s.io/kind@v0.17.0
----

. Check installed version
+
[source,console]
----
$ $(go env GOPATH)/bin/kind version
kind v0.17.0 go1.19 linux/amd64
----

=== Install Ingress

. Create Cluster +
Create a kind cluster with extraPortMappings and node-labels.
+
--
[horizontal]
extraPortMappings:: allow the local host to make requests to the Ingress controller over ports 80/443
node-labels:: only allow the ingress controller to run on a specific node(s) matching the label selector
--
+
[source,shell]
----
#!/bin/bash
cat <<EOF | kind create cluster --config=-
kind: Cluster
apiVersion: kind.x-k8s.io/v1alpha4
nodes:
- role: control-plane
  kubeadmConfigPatches:
  - |
    kind: InitConfiguration
    nodeRegistration:
      kubeletExtraArgs:
        node-labels: "ingress-ready=true"
  extraPortMappings:
  - containerPort: 30000
    hostPort: 30000
    protocol: TCP
  - containerPort: 30080
    hostPort: 30080
    protocol: TCP
  - containerPort: 30081
    hostPort: 30081
    protocol: TCP
EOF
----

. Install Contour
+
[source,shell]
----
kubectl apply -f https://projectcontour.io/quickstart/contour.yaml
----

. Apply it by running:
+
Apply kind specific patches to forward the hostPorts to the ingress controller, set taint tolerations and schedule it to the custom labelled node.
+
[source,json]
----
{
  "spec": {
    "template": {
      "spec": {
        "nodeSelector": {
          "ingress-ready": "true"
        },
        "tolerations": [
          {
            "key": "node-role.kubernetes.io/control-plane",
            "operator": "Equal",
            "effect": "NoSchedule"
          },
          {
            "key": "node-role.kubernetes.io/master",
            "operator": "Equal",
            "effect": "NoSchedule"
          }
        ]
      }
    }
  }
}
----
+
[source,shell]
----
kubectl patch daemonsets -n projectcontour envoy -p '{"spec":{"template":{"spec":{"nodeSelector":{"ingress-ready":"true"},"tolerations":[{"key":"node-role.kubernetes.io/control-plane","operator":"Equal","effect":"NoSchedule"},{"key":"node-role.kubernetes.io/master","operator":"Equal","effect":"NoSchedule"}]}}}}'
----

. Make sure the EXTERNAL-IP value is `<none>`
+
[source,console]

NAME         TYPE        CLUSTER-IP   EXTERNAL-IP   PORT(S)   AGE
kubernetes   ClusterIP   10.96.0.1    <none>        443/TCP   15m

. Deploy a app
+
[source,yaml]
.greet-go-app.loadBalancerIP.yaml
----
apiVersion: v1
kind: Service
metadata:
  name: greet-go-app-metallb
spec:
  type: LoadBalancer
  loadBalancerIP: 172.18.0.240 # <1>
  selector:
    app: greet-go-app
  ports:
    - name: http
      port: 8080
      targetPort: 808
