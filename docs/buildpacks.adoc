=== Install pack

* https://buildpacks.io/docs/tools/pack/[Pack · Cloud Native Buildpacks^]

.Install on Ubuntu

. Install `pack-cli`
+
[source,shell]
----
sudo add-apt-repository ppa:cncf-buildpacks/pack-cli
sudo apt-get update
sudo apt-get install pack-cli
----

. Auto-completion
+
[source,shell]
.bashrc
----
type -p pack > /dev/null && . $(pack completion)
----
